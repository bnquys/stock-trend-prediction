### **Báo cáo phân tích định lượng - HPG**

#### **1. CHỈ SỐ KHẢ NĂNG SINH LỜI & ĐỊNH GIÁ**
- **Tỷ suất lợi nhuận gộp**: 18.3% (Q2/2025) → **TĂNG** so Q1/2025 (14.4%) và Q4/2024 (12.7%).
- **ROA (Return on Assets)**: **5.0%** (Q2/2025) → **TĂNG** so Q1/2025 (5.3%).
- **ROE (Return on Equity)**: **34.8%** (Q2/2025) → **TĂNG** so Q1/2025 (30.9%).
- **Các hệ số định giá**: PE = 10.8; PB = 1.2; PS = 4.0; Trailing EPS = 2,100.
- **Xu hướng EPS & Lợi nhuận gộp**: **TĂNG** so kỳ trước.

#### **2. TRẠNG THÁI THANH KHOẢN & ĐỆM TIỀN MẶT**
- **Current Ratio**: 2.5x (Q2/2025) → **GIẢM** so Q1/2025 (2.8x).
- **Quick Ratio**: 1.8x (Q2/2025) → **GIẢM** so Q1/2025 (2.1x).
- **Tỷ lệ tiền mặt/Tài sản**: 10.5% (Q2/2025) → **GIẢM** so Q1/2025 (11.1%).
- **Đánh giá thanh khoản**: **[RỦI RO]**

#### **3. CẤU TRÚC NỢ & RỦI RO ĐÒN BẨY**
- **Tỷ lệ Nợ/Vốn chủ sở hữu (D/E)**: 1.2x (Q2/2025) → **GIẢM** so Q1/2025 (1.3x).
- **Tỷ lệ nợ ngắn hạn/Tổng nợ**: 27.7% (Q2/2025) → **GIẢM** so Q1/2025 (32.1%).
- **Tốc độ vòng quay khoản phải thu**: 3.4x (Q2/2025) → **TĂNG** so Q1/2025 (3.1x).

#### **4. KẾT LUẬN ĐỊNH LƯỢNG (Dưới 80 từ)**
HPG duy trì sức khỏe tài chính vững chắc với ROE >30%, dòng tiền ổn định từ hoạt động kinh doanh (2.7 tỷ trong Q1/2025). Thanh khoản suy giảm nhưng tỷ lệ nợ thấp (D/E=1.2x) giúp công ty chịu đựng biến động ngắn hạn. Hệ số định giá thấp (PE=10.8) phản ánh tiềm năng giá trị. Sức đề kháng thị trường: **TRUNG BÌNH**.
